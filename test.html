1. Series Methods

A Series is a one-dimensional labeled array capable of holding any data type.

Creation

pd.Series(data, index=None, dtype=None, name=None, copy=False)

s = pd.Series([1, 2, 3], index=['a', 'b', 'c'], name="example")

Attributes

  • s.index: Access the index (labels) of the Series.
  • s.values: Return data as a NumPy array.
  • s.name: Get or set the Series name.
  • s.dtype: Data type of elements.

Basic Methods

  • s.head(n): First n elements (default 5).
  • s.tail(n): Last n elements (default 5).
  • s.size: Number of elements.
  • s.shape: Shape of the Series (length,).

Statistical Operations

  • Aggregations: s.sum(), s.mean(), s.median(), s.std(), s.var()
  • Cumulative: s.cumsum(), s.cumprod()
  • Min/Max: s.min(), s.max()

Logical Operations

  • s.any(): Returns True if any element is True.
  • s.all(): Returns True if all elements are True.

String Operations

Applicable when dtype is str:

  • s.str.upper(), s.str.lower(): Change case.
  • s.str.contains('pattern'): Check substring presence.
  • s.str.len(): Length of strings.

Manipulation

  • s.append(other): Concatenate two Series.
  • s.drop(labels): Remove specified labels.
  • s.sort_values(): Sort elements by value.
  • s.rank(): Rank elements.

2. DataFrame Methods

A DataFrame is a two-dimensional labeled data structure.

Creation

pd.DataFrame(data, index=None, columns=None, dtype=None, copy=False)

data = {'Name': ['Alice', 'Bob'], 'Age': [25, 30]}
df = pd.DataFrame(data)

Attributes

  • df.columns: Column labels.
  • df.index: Row labels.
  • df.dtypes: Data types of each column.
  • df.shape: Tuple (rows, columns).

Inspection

  • df.info(): Overview of the DataFrame.
  • df.describe(): Statistical summary of numerical columns.
  • df.head(n): First n rows.
  • df.tail(n): Last n rows.
